# 독도 생태 아카이브

독도의 육상 식물, 곤충과 조류, 해양 생물, 해양 환경과 보전 과제를 공공기관 자료를 바탕으로 소개하는 반응형 교육 웹사이트입니다.

## 주요 기능

- 생물 분류 필터: 식물 / 곤충·조류 / 해양생물
- 해양 생태계의 층위와 2020년 조사 결과 시각화
- 독도 기후와 보전 과제 안내
- 3문항 인터랙티브 생태 퀴즈
- 모바일 메뉴, 키보드 접근성, 모션 감소 설정 지원

## 기술

별도 프레임워크 없이 HTML, CSS, JavaScript로 제작했습니다.
